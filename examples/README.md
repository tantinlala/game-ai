# Example Games

This directory contains example game files in both strategic form (.nfg) and extensive form (.efg) formats.

## Strategic Form Games (.nfg)

### Prisoner's Dilemma
A classic 2-player game demonstrating the conflict between individual and collective rationality.

### Battle of the Sexes
A coordination game with two Nash equilibria.

### Matching Pennies
A zero-sum game with only a mixed strategy Nash equilibrium.

## Extensive Form Games (.efg)

### Sequential Entry Game
A simple sequential game with perfect information.

### Poker (Simplified)
A simplified poker game with imperfect information.

## Using Example Games

You can load these games in the application by:

1. Copying the content into the editor pane
2. Using `/validate` to check the file
3. Using `/solve` to compute Nash equilibria
4. Modifying values and resolving

Or use them as templates when building new games with the AI assistant.
